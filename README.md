# 2006 Revival - Launcher & Client Distribution

This repo contains the **launcher** and **client runtime files** for 2006 Revival.

No server source code is included here — this is the public download repo for friends.

---

## Download

### Full Launcher (recommended)
The full launcher has the client embedded — just download and double-click:

**[Download Tristans2006Launcher.jar](https://github.com/tristan9988/tristans-2006-launcher/raw/main/dist/Tristans2006Launcher.jar)**

### Shareable Launcher (smaller)
A smaller version that downloads the client on first run:

**[Download Tristans2006Launcher-shareable.jar](https://github.com/tristan9988/tristans-2006-launcher/raw/main/dist/Tristans2006Launcher-shareable.jar)**

---

## How to Play

1. Download `Tristans2006Launcher.jar` from above
2. Double-click it (requires Java installed)
3. The launcher auto-connects to the game server
4. Click **PLAY**

If the full jar is too big for Discord, use the shareable version. On first launch it will
download the client automatically.

---

## Auto-Updates

The launcher checks this repo for client updates on every launch.
When the game client is updated, your launcher will pull the changes automatically
(requires [Git](https://git-scm.com) installed for auto-updates).
